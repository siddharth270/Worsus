# Worsus

## Project Description
Worsus is an engaging game that challenges players with strategic thinking and quick decision-making. Players must navigate through various levels, overcoming obstacles and competing against opponents.

## Features
- **Multi-level gameplay:** Each level offers unique challenges and puzzles.
- **Real-time multiplayer:** Compete against players from around the world.
- **Customizable characters:** Personalize your gaming experience with a variety of character options.
- **Leaderboards:** Track your progress and compete for the top spots.

## Tech Stack
- **Frontend:** React.js for building user interfaces.
- **Backend:** Node.js with Express.js for server-side logic.
- **Database:** MongoDB for data storage.
- **Game Engine:** Phaser.js for 2D game development.

## Setup Instructions
1. **Clone the repository:**
   ```bash
   git clone https://github.com/siddharth270/Worsus.git
   ```
2. **Navigate to the project directory:**
   ```bash
   cd Worsus
   ```
3. **Install dependencies:**
   ```bash
   npm install
   ```
4. **Run the application:**
   ```bash
   npm start
   ```

## How to Play
- Use arrow keys to move your character.
- Collect items to gain points and bonuses.
- Avoid obstacles and defeat opponents to progress through levels.
